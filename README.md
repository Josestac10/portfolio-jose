# Portfolio de Jose Santa Cruz - Opcion 2

Segunda propuesta visual para el portfolio personal. Esta version usa una estetica tipo CV digital / tablero creativo: fondo claro, tarjetas con bordes marcados, sombras tipo sticker y una presentacion mas cercana, juvenil y editable.

## Como ejecutar

```bash
npm install
npm run dev
```

Luego abre la URL que aparezca en la terminal, normalmente `http://localhost:5173`.

## Como editar tu informacion

La informacion principal esta en:

```txt
src/data/profile.js
```

Ahi puedes cambiar:

- Nombre, rol, ubicacion y descripcion personal.
- Correo, GitHub, LinkedIn y CV.
- Habilidades.
- Proyectos.
- Certificados.
- Temas que estas aprendiendo actualmente.

## Diferencia con la primera opcion

- Esta version se siente mas clara, tipo hoja de presentacion moderna.
- Usa fondo claro, tarjetas grandes y bordes visibles.
- Puede funcionar mejor si quieres transmitir cercania, orden y perfil junior honesto.
- La primera opcion tiene un estilo mas futurista/dark; esta opcion es mas editorial y amigable.
